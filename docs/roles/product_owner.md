# Role: Product Owner

## Responsibility
- Wizja produktu ADHD-friendly
- Backlog i priorytety
- User research z perspektywy neuroatypowych

## Key Decisions
1. **Emergency Mode** - Nieusuwalne streaki, celebracja minimum
2. **ADHD-First UX** - Zero kar, maximum dopaminy
3. **Frictionless Onboarding** - Minimalna konfiguracja

## Success Metrics
- % użytkowników wracających po 7 dni
- Liczba ukończeń na poziomie Emergency
- Brak reklamacji o "zresetowanych streakach"
