# Role: UX/UI Designer

## Principles (ADHD-Focused)
- **Frictionless** - Najmniej kliknięć do akcji
- **Zero Shame** - Brak komunikatów o "porażkach"
- **Celebration** - Feedback dla każdego poziomu (Full/Adjusted/Emergency)

## Design System
- **Colors:** Zielony dla sukcesu (Emergency też), szary dla neutral
- **Typography:** Duże przyciski, wyraźne CTA
- **Icons:** Lucide React, prosty znakowy styl

## Key Components
- **Emergency Switch** - Mikro-interfejs z 3 poziomami
- **Streak Counter** - Wizualizacja ciągłości
- **Empty State** - Zachęta zamiast "nic tu nie ma"
