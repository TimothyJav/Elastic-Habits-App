# UX/UI Designer

## Odpowiedzialność
- Projektowanie interfejsu mobile-first.
- Redukcja tarcia w codziennym użyciu aplikacji.
- Pilnowanie zasad no-shame UX.

## Zasady ADHD-Friendly
- **Frictionless** - najmniej kliknięć do wykonania akcji.
- **Zero Shame** - brak komunikatów sugerujących porażkę.
- **Celebration** - pozytywny feedback dla poziomów Full, Adjusted i Emergency.

## System projektowy
- **Kolory:** zielony dla sukcesu, także na poziomie Emergency.
- **Typografia:** duże przyciski i wyraźne CTA.
- **Ikony:** Lucide React, prosty znakowy styl.

## Kluczowe komponenty
- **Emergency Switch** - mikrointerfejs z trzema poziomami wykonania.
- **Streak Counter** - wizualizacja ciągłości bez mechanizmu kary.
- **Empty State** - zachęta zamiast pustego, oceniającego komunikatu.
